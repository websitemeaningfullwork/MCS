# **MCA Homepage Redesign Documentation (Front Page Only)**

## **Objective**

The homepage should be redesigned to match the premium reference design provided by the client.

The overall goal is to create a modern, premium, mentorship-first homepage with excellent visual hierarchy, clean spacing, colorful accents, and a highly engaging first impression.

The new homepage should closely follow the provided reference image.

---

# **1\. Hero Section (Landing Screen)**

This will be the very first section users see after entering the website.

This hero section should follow the reference design almost exactly.

Requirements:

* Large full-width hero background image.  
* Student image positioned on the right side.  
* Hero heading positioned on the left side.  
* Floating navigation bar should overlap the hero image.  
* The hero section should feel immersive and premium.  
* Keep the overall composition and proportions similar to the provided reference.

Do not redesign this section differently.

The reference layout should be followed as closely as possible.

---

# **2\. Navigation Bar**

The navigation bar should remain floating on top of the hero image.

Style Requirements

* Glassmorphism effect  
* Soft blur background

* Semi-transparent white background  
* Smooth hover animations  
* Premium spacing

Navigation Items

* Home  
* Programs  
* Mentors  
* E-Books  
* Live Classes

Search bar should remain on the right.

Notification icon should remain.

User profile should remain.

Logo will be added later by the client, so leave the logo area ready without changing the layout.

Immediately below the hero section, keep the colorful feature cards exactly like the reference.

Cards:

* Live Classes → Green  
* Ask Mentor → Orange

The overall appearance should remain colorful, balanced, and premium.

# **6\. Continue Your Journey**

Keep this section exactly as shown in the reference.

Requirements

* Continue Learning card  
* Course thumbnail

# **10\. Remove Large Empty Spaces**

The current homepage contains large empty white spaces between sections.

These empty spaces make the website feel unfinished.

Avoid leaving unnecessary blank areas.

The homepage should feel visually rich from top to bottom.

Each section should naturally connect to the next.

---

# **11\. Color System**

Avoid making the interface entirely green.

Use a premium colorful design language.

Primary colors

* Sky Blue  
* Royal Blue

# **12\. Design Language**

The homepage should feel

* Premium  
* Modern  
* Clean  
* Friendly  
* Inspiring

❌ Corporate Dashboard

---

# **13\. Spacing & Visual Balance**

Use a consistent spacing system throughout the homepage.

Recommended spacing scale:

* 16px  
* 24px  
* 32px  
* 48px  
* 64px

Avoid random spacing.

Every section should maintain proper visual rhythm.

---

# **14\. Animations**

Use elegant micro-interactions only.

Include:

* Smooth hover animations  
* Card elevation  
* Soft shadow transitions  
* Fade-in effects  
* Button ripple  
* Glass reflection  
* Smooth page transitions

Avoid flashy or distracting animations.

---

## **Final Goal**

The homepage should closely match the provided reference while preserving MCA's branding and functionality.

The final result should immediately communicate that MCA is a premium mentorship platform rather than a generic online course website.

The homepage should feel modern, colorful, trustworthy, visually balanced, and professionally designed, with no unnecessary empty spaces or outdated sections.

## **Page Name**

**Secure Checkout – MCA (Meaningful Career Academy)**

---

# **Objective**

The checkout page must feel **premium, trustworthy, modern, clean, spacious, and extremely easy to understand**.

The user should never feel confused after arriving on this page.

The design should reduce payment anxiety and increase conversion rate.

The page should follow the exact premium design language used on the MCA Homepage.

---

# **Overall Design Language**

* Premium White / Creamy White background  
* Large amount of white space  
* Soft glassmorphism  
* Rounded corners (20–24px)  
* Smooth shadows  
* Clean layout  
* Modern Apple / Stripe / Linear inspired interface  
* Minimal but premium  
* No clutter  
* No unnecessary text

Everything should feel soft and elegant.
